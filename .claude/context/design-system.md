# Design System — Create with Anmol

# Design Philosophy

Create with Anmol follows a:

* premium
* cinematic
* editorial
* Apple Liquid Glass inspired

visual system.

The interface should feel:

* sophisticated
* modern
* restrained
* atmospheric
* immersive
* creator-focused

The design should prioritize:

* clarity
* hierarchy
* readability
* spacing
* subtle depth
* premium material behavior

The experience should never feel:

* flashy
* noisy
* gimmicky
* cyberpunk
* gaming-oriented
* overdesigned

# Background System

The website uses:

# layered atmospheric backgrounds.

The background should:

* remain dark overall
* avoid flat black emptiness
* create subtle environmental depth
* support content readability
* feel visually rich without distraction

Preferred background direction:

```css id="7l6o2r"
background:
radial-gradient(
  circle at top,
  #1f2329 0%,
  #13161b 35%,
  #0b0d10 70%,
  #07090c 100%
),
radial-gradient(
  circle at 20% 20%,
  rgba(180,190,210,0.05),
  transparent 30%
),
radial-gradient(
  circle at 80% 30%,
  rgba(255,255,255,0.03),
  transparent 25%
);
```

Optional:

* extremely subtle noise texture overlay
* low-opacity atmospheric lighting

Avoid:

* colorful gradients
* neon diffusion
* strong blue/purple lighting
* visible mesh gradients
* animated backgrounds

# Glass Material System

Glass effects should remain:

* restrained
* subtle
* premium

The system should avoid:

* excessive blur
* heavy transparency
* over-layered glass effects

Preferred glass styling:

```css id="jlx1k6"
background: rgba(255,255,255,0.06);
border: 1px solid rgba(255,255,255,0.08);
backdrop-filter: blur(12px);
```

Glass surfaces should feel:

* lightweight
* floating
* tactile
* cinematic

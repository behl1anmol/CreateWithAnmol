# Vision — Create with Anmol

## Product Identity

Create with Anmol is a premium cinematic creator platform focused on:

* AI creativity
* prompt engineering
* cinematic AI workflows
* technical content creation
* creator education
* digital creator products

The platform acts as a central creator ecosystem that connects:

* Instagram content
* AI prompt packs
* Gumroad digital products
* Medium technical blogs
* future YouTube educational content

The website is not intended to behave like:

* a SaaS dashboard
* a startup landing page
* a portfolio template
* a generic blog platform

Instead, it should feel like:

* a premium creator operating system
* an editorial creator platform
* a cinematic AI creator brand

---

# Core Product Goals

The platform exists to:

1. Build creator authority
2. Increase Instagram engagement
3. Showcase AI prompt workflows
4. Drive Gumroad product discovery
5. Establish a recognizable creator identity
6. Create a premium visual experience
7. Build long-term audience trust

# Visual Direction

The visual identity follows:

* Apple Liquid Glass inspired aesthetics
* cinematic editorial interfaces
* dark graphite environments
* restrained glassmorphism
* premium typography
* atmospheric depth

The design language should emphasize:

* elegance
* restraint
* readability
* subtle depth
* modern premium minimalism

Avoid:

* excessive glow
* neon effects
* cyberpunk aesthetics
* cluttered UI
* overly futuristic visuals
* heavy gaming influences

---

# Content Philosophy

Content should feel:

* curated
* intentional
* premium
* visually cohesive

The platform should avoid:

* cluttered discovery systems

Featured content should feel:

* selected
* important
* visually elevated

* smooth
